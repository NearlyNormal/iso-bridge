    About ISO Bridge

    ISO Bridge is a country-aware payments format translation workbench. It converts between SWIFT MT legacy messages and ISO 20022 XML, showing exactly what happens to every field in the process.

    The global payments industry is migrating from SWIFT MT to ISO 20022, but the transition is uneven: different countries, different rails, different timelines. Translation between formats is inherently lossy — ISO 20022 carries richer data than MT can represent, and MT fields don't always have clean ISO equivalents.

    This tool is honest about data loss. Every translation shows:

    • Which fields map cleanly (green)
    • Which fields are transformed (amber)
    • Which data is lost or can't be represented (red)
    • Which ISO fields are auto-generated with no MT source (blue)

    v1.0 coverage: MT103 ↔ pacs.008 translation across US, Canada, and Brazil. More formats and countries in v1.1.

    Part of the payments toolkit:

    • SWIFT MT Parser — decode MT messages into structured JSON
    • ISO 20022 Validator — validate XML against schemas and scheme profiles
    • ISO Bridge (this tool) — translate between formats
